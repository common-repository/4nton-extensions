== Description ==

Anton Extensions has PHP coding SOP fuction that prevent errors, list of addons that may suitable for your site requirements, and many more free features that you might love. This plugin is can integrate with Gravity Forms, Override WordPress default emails, login landing page, and more.

= FREE Features =
* Manage wp-login.php login landing page. Override the default WordPress layout, logo and permalink. [DEMO](https://demo.anthonycarbon.com/wp-login.php)
* Override the WordPress email subject name and email using general settings name and email address.
* Turn your Gravity Form activation page into popup layout with admin settings.
* Override the WordPress new user registration subject name ([%s] New User Registration).
* Disable/Enable input auto fill or autocomplete.
* Fix Gravity Forms Uncaught TypeError: grecaptcha.render


= Addons =
* [Gravity Forms Media Upload Field](https://www.anthonycarbon.com/product/gravity-forms-media-upload-field/) - add custom field that can upload image with prefered dimension, save to media dashboard.
* [WP Print](https://www.anthonycarbon.com/product/wp-print/) - Customized your post, page, or custom post-type print layout without doing any back-end coding.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the `Plugins` screen in WordPress
3. Use the **Anton Extensions** menu in your dashboard.
